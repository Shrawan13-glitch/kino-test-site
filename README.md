# Kino – AI Assistant

Welcome to the official website source for **Kino**, the helpful AI assistant you’re chatting with right now.

## What is Kino?
- A conversational AI built on the latest large‑language‑model technology.
- Integrated with a rich toolbox: web search, PDF generation, speech synthesis, and full GitHub automation.
- Designed to be **friendly, transparent, and extensible** – you can add new tools, customize prompts, or embed Kino into apps.

## Features Showcase
| Category | Capability |
|----------|-------------|
| **Information** | Real‑time web search, summarisation, fact‑checking |
| **Content Creation** | Generate PDFs, HTML, markdown, code snippets |
| **Audio** | Text‑to‑speech with multiple voices |
| **Automation** | Manage GitHub repos, branches, PRs, issues, Actions |
| **File System** | Virtual file system for temporary storage |
| **Extensibility** | Add custom tools via JSON config |

## Quick Demo (HTML)
Visit the live site (hosted via GitHub Pages) to see a live demo of Kino’s abilities:
- **Ask questions** – type a query and get an answer.
- **Generate PDFs** – click a button to download a sample PDF.
- **Listen** – press play to hear a text‑to‑speech example.

## How to Deploy
1. Fork this repository.
2. Enable **GitHub Pages** in Settings → Pages → source `main` branch.
3. The site will be available at `https://<your‑username>.github.io/kino-test-site/`.
