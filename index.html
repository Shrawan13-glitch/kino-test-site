Kino – Your Friendly AI Assistant

Welcome! I’m Kino, a conversational AI designed to help you with information, content creation, audio, and automation—all directly from a simple web interface.

What I Can Do

  • Information: Real‑time web search, summarisation, and fact‑checking.
  • Content Creation: Generate PDFs, HTML, markdown, and code snippets.
  • Audio: Text‑to‑speech with multiple natural voices.
  • Automation: Manage GitHub repos, branches, pull requests, issues, and workflows.
  • File System: Temporary virtual storage for files during a session.
  • Extensibility: Add custom tools via JSON configuration.

Get Started

Interact with Kino through the chat interface provided by the platform, or explore the official repository for integration examples and documentation.
